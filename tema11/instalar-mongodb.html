Instalación de MongoDB en Ubuntu 24.04

A continuación se detallan los pasos necesarios para instalar MongoDB 7.0 en Ubuntu 24.04. Sigue cada paso en orden para completar la instalación correctamente.

1. Importar la clave pública de MongoDB

El primer paso es importar la clave GPG oficial de MongoDB para verificar la autenticidad de los paquetes:

curl -fsSL https://pgp.mongodb.com/server-7.0.asc | sudo gpg -o /usr/share/keyrings/mongodb-server-7.0.gpg --dearmor

2. Crear el archivo de repositorio

Añade el repositorio oficial de MongoDB a las fuentes de APT:

echo "deb [ arch=amd64,arm64 signed-by=/usr/share/keyrings/mongodb-server-7.0.gpg ] https://repo.mongodb.org/apt/ubuntu jammy/mongodb-org/7.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-7.0.list

3. Actualizar los paquetes

Actualiza la lista de paquetes disponibles para incluir el repositorio de MongoDB:

sudo apt update

4. Instalar MongoDB

Instala MongoDB y todos sus componentes necesarios:

sudo apt install mongodb-org

5. Iniciar el servicio de MongoDB

Inicia el servicio de MongoDB utilizando systemctl:

sudo systemctl start mongod

6. Habilitar MongoDB para que inicie con el sistema

Configura MongoDB para que se inicie automáticamente al arrancar el sistema:

sudo systemctl enable mongod

7. Verificar que MongoDB esté funcionando

Comprueba el estado del servicio para asegurarte de que MongoDB está ejecutándose correctamente:

sudo systemctl status mongod

8. Conectarse a MongoDB

Para acceder a la shell interactiva de MongoDB, utiliza el comando mongosh:

mongosh

9. Verificar la versión instalada

Confirma la versión de MongoDB que has instalado:

mongod --version

¡Instalación completada! Ahora tienes MongoDB 7.0 funcionando en tu sistema Ubuntu 24.04.
