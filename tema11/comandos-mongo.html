Comandos Básicos de MongoDB Shell (mongosh)

MongoDB Shell (mongosh) es la interfaz de línea de comandos interactiva para trabajar con MongoDB. A continuación se presentan los comandos más utilizados organizados por categorías.

1. Conexión a MongoDB

Para conectarte a una instancia de MongoDB, utiliza los siguientes comandos:

mongosh                    # Conectar a localhost en puerto predeterminado
mongosh "mongodb://host:puerto"   # Conectar a un host específico

2. Bases de datos

Comandos para gestionar bases de datos:

show dbs                   // Mostrar todas las bases de datos
use nombreDB               // Crear/cambiar a una base de datos
db                        // Mostrar base de datos actual
db.dropDatabase()         // Eliminar base de datos actual

3. Colecciones

Las colecciones son equivalentes a las tablas en bases de datos relacionales. Comandos básicos:

show collections          // Mostrar todas las colecciones
db.createCollection("nombre")     // Crear una colección
db.nombreColeccion.drop()        // Eliminar una colección

4. Insertar documentos

Para añadir documentos (registros) a una colección:

db.coleccion.insertOne({campo: "valor"})    // Insertar un documento

db.coleccion.insertMany([                   // Insertar varios documentos
  {campo1: "valor1"},
  {campo2: "valor2"}
])

5. Consultar documentos

Comandos para recuperar información de las colecciones:

db.coleccion.find()                     // Encontrar todos los documentos
db.coleccion.find().pretty()            // Mostrar resultados formateados
db.coleccion.findOne()                  // Encontrar un documento
db.coleccion.find({campo: "valor"})     // Buscar por criterio

6. Operadores de consulta

MongoDB proporciona operadores especiales para realizar consultas más complejas:

db.coleccion.find({campo: {$gt: 20}})   // Mayor que
db.coleccion.find({campo: {$lt: 20}})   // Menor que
db.coleccion.find({campo: {$in: [1,2,3]}}) // Valores en array
db.coleccion.find({$or: [{a:1}, {b:2}]})   // Condición OR

7. Actualizar documentos

Para modificar documentos existentes en una colección:

db.coleccion.updateOne(                  // Actualizar un documento
  {filtro: "valor"},
  {$set: {campo: "nuevo_valor"}}
)

db.coleccion.updateMany(                 // Actualizar varios documentos
  {filtro: "valor"},
  {$set: {campo: "nuevo_valor"}}
)

8. Eliminar documentos

Comandos para borrar documentos de una colección:

db.coleccion.deleteOne({campo: "valor"})    // Eliminar un documento
db.coleccion.deleteMany({campo: "valor"})   // Eliminar varios documentos

9. Límites y ordenamiento

Controla la cantidad y el orden de los resultados de tus consultas:

db.coleccion.find().limit(5)            // Limitar resultados
db.coleccion.find().sort({campo: 1})    // Ordenar ascendente
db.coleccion.find().sort({campo: -1})   // Ordenar descendente

10. Agregaciones básicas

Operaciones para obtener información estadística de las colecciones:

db.coleccion.count()                    // Contar documentos
db.coleccion.distinct("campo")          // Valores únicos

11. Índices

Los índices mejoran el rendimiento de las consultas. Comandos para gestionarlos:

db.coleccion.createIndex({campo: 1})    // Crear índice ascendente
db.coleccion.getIndexes()               // Ver índices existentes
db.coleccion.dropIndex("nombre_index")  // Eliminar índice
Nota: Recuerda que MongoDB es una base de datos NoSQL orientada a documentos. Los documentos se almacenan en formato BSON (Binary JSON) y no requieren un esquema fijo, lo que proporciona gran flexibilidad en el modelado de datos.
