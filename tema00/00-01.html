Historia de Python

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, creado por Guido van Rossum a finales de los años 80 en el Centro para las Matemáticas y la Informática (CWI) en los Países Bajos.

Origen del nombre

El nombre "Python" no proviene de la serpiente, sino del grupo de comedia británico "Monty Python's Flying Circus", del cual Guido van Rossum era fanático.

Cronología de Python

  • 1989: Guido van Rossum comienza el desarrollo de Python como sucesor del lenguaje ABC
  • 1991: Se publica Python 0.9.0, que incluía clases con herencia, manejo de excepciones, funciones y módulos
  • 1994: Python 1.0 introduce programación funcional con lambda, map, filter y reduce
  • 2000: Python 2.0 añade comprensiones de listas y recolección de basura
  • 2008: Python 3.0 (también conocido como "Python 3000") rompe la compatibilidad hacia atrás para corregir defectos fundamentales del lenguaje
  • 2020: Fin del soporte para Python 2.7
  • Actualidad: Python sigue evolucionando con nuevas versiones anuales
Filosofía de Python - El Zen de Python

Python sigue una filosofía de diseño que enfatiza la legibilidad del código. Puedes verla ejecutando import this en el intérprete de Python.

Principales Diferencias con Java

Aunque tanto Python como Java son lenguajes orientados a objetos y multiplataforma, presentan diferencias fundamentales en su filosofía y sintaxis:

Comparación de Sintaxis Python vs Java 
# Python - Hola Mundo
print("¡Hola, mundo!")

// Java - Hola Mundo
public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("¡Hola, mundo!");
    }
}

Diferencias Clave

1. Tipado

  • Python: Tipado dinámico - no necesitas declarar el tipo de las variables
  • Java: Tipado estático - debes declarar el tipo de cada variable

2. Compilación vs Interpretación

  • Python: Interpretado - el código se ejecuta línea por línea
  • Java: Compilado a bytecode - se compila antes de ejecutar

3. Sintaxis

  • Python: Usa indentación para delimitar bloques de código
  • Java: Usa llaves {} y punto y coma obligatorio

4. Velocidad

  • Python: Generalmente más lento debido a su naturaleza interpretada
  • Java: Más rápido en ejecución gracias a la JVM

5. Curva de Aprendizaje

  • Python: Más fácil de aprender, sintaxis más natural
  • Java: Más verboso, requiere más conceptos iniciales
