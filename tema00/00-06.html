Jupyter Notebooks

Documentos interactivos para desarrollo, análisis y prototipado en Python

¿Qué son los Jupyter Notebooks?

Jupyter Notebooks son documentos interactivos que combinan código ejecutable, texto explicativo, visualizaciones y otros elementos multimedia en un solo lugar. Son especialmente populares en ciencia de datos, investigación, educación y prototipado rápido.

¿Por qué usar Jupyter Notebooks?

  • Interactividad: Ejecuta código celda por celda y ve los resultados inmediatamente
  • Documentación integrada: Combina código con explicaciones en texto enriquecido (Markdown)
  • Visualización: Ideal para mostrar gráficos, tablas y resultados de análisis
  • Prototipado rápido: Perfecto para experimentar y probar ideas paso a paso
  • Compartir resultados: Fácil de compartir análisis completos con otros
  • Reproducibilidad: Documenta todo el proceso de análisis de principio a fin
Casos de uso comunes

Jupyter es especialmente útil para análisis de datos, machine learning, visualización, investigación científica, educación, prototipado de algoritmos y creación de reportes interactivos.

Estructura de un Notebook

Un Jupyter Notebook está compuesto por celdas que pueden contener:

  • Código Python: Que se ejecuta y muestra resultados
  • Texto Markdown: Para documentación, explicaciones y títulos
  • HTML crudo: Para elementos más avanzados
  • Salidas: Resultados de ejecución, gráficos, tablas, etc.

Diferencias entre herramientas Jupyter

  • Jupyter Notebook: Interfaz clásica, simple y enfocada en un solo notebook
  • JupyterLab: Interfaz moderna tipo IDE con pestañas, explorador de archivos y extensiones
  • VS Code + Jupyter: Extensión que integra notebooks en VS Code con todas sus ventajas

Instalación de Jupyter en Windows

Existen varias formas de usar Jupyter en Windows. Te mostraremos las opciones más populares:

Método 1: Extensión de Jupyter para VS Code (Recomendado)

La forma más moderna y conveniente de usar Jupyter es a través de Visual Studio Code:

  1. Instalar VS Code: Descarga desde code.visualstudio.com
  2. Instalar la extensión Python: Busca "Python" en el marketplace de extensiones
  3. Instalar la extensión Jupyter: Busca "Jupyter" en el marketplace de extensiones
  4. Configurar el intérprete de Python: Ctrl+Shift+P → "Python: Select Interpreter"
Instalar Jupyter desde PowerShell PowerShell 
# 1. Crear y activar entorno virtual:
python -m venv jupyter_env
jupyter_env\Scripts\activate

# 2. Actualizar pip:
python -m pip install --upgrade pip

# 3. Instalar Jupyter:
pip install jupyter jupyterlab

# 4. Instalar paquetes útiles para ciencia de datos:
pip install numpy pandas matplotlib seaborn plotly

# 5. Verificar la instalación:
jupyter --version

Crear tu primer Notebook en VS Code

  1. Abre VS Code
  2. Crea un nuevo archivo con extensión .ipynb (ej: mi_notebook.ipynb)
  3. VS Code detectará automáticamente que es un Jupyter Notebook
  4. Selecciona el kernel de Python (tu entorno virtual)
  5. ¡Empieza a escribir código en las celdas!
Ventajas de VS Code + Jupyter

VS Code con Jupyter ofrece las mejores características de ambos mundos: la potencia de un IDE moderno con la interactividad de los notebooks, incluyendo IntelliSense, debugging, control de versiones integrado y más.

Método 2: Instalación tradicional de Jupyter

También puedes usar Jupyter en el navegador web de forma tradicional:

Instalar y ejecutar Jupyter tradicional PowerShell 
# Con el entorno virtual activado:
pip install jupyter

# Iniciar Jupyter Notebook:
jupyter notebook

# O iniciar JupyterLab (interfaz moderna):
pip install jupyterlab
jupyter lab

# El navegador se abrirá automáticamente en http://localhost:8888

Instalación de Jupyter en WSL

Usar Jupyter en WSL te permite aprovechar el entorno Linux para análisis de datos mientras mantienes la comodidad de Windows:

Configuración de Jupyter Server en WSL

Esta configuración te permite ejecutar Jupyter en WSL y acceder desde Windows:

Instalar Jupyter en WSL WSL Terminal 
# 1. Actualizar el sistema:
sudo apt update && sudo apt upgrade -y

# 2. Crear entorno virtual:
python3 -m venv jupyter_env
source jupyter_env/bin/activate

# 3. Instalar Jupyter y dependencias:
pip install --upgrade pip
pip install jupyter jupyterlab
pip install numpy pandas matplotlib seaborn plotly scikit-learn

# 4. Verificar instalación:
jupyter --version

Configurar Jupyter para acceso desde Windows

Para acceder al servidor Jupyter de WSL desde Windows, necesitas configurar algunos parámetros:

Generar configuración de Jupyter WSL Terminal 
# Generar archivo de configuración:
jupyter notebook --generate-config

# El archivo se creará en ~/.jupyter/jupyter_notebook_config.py

# Generar contraseña (opcional pero recomendado):
jupyter notebook password

Editar el archivo de configuración para permitir acceso desde Windows:

Configurar acceso remoto WSL Terminal 
# Editar archivo de configuración:
nano ~/.jupyter/jupyter_notebook_config.py

# Añadir o modificar estas líneas:
c.NotebookApp.ip = '0.0.0.0'
c.NotebookApp.port = 8888
c.NotebookApp.open_browser = False
c.NotebookApp.allow_remote_access = True

Iniciar Jupyter Server

Ejecutar Jupyter en WSL WSL Terminal 
# Activar entorno virtual:
source jupyter_env/bin/activate

# Iniciar Jupyter Notebook:
jupyter notebook

# O iniciar JupyterLab:
jupyter lab

# Acceder desde Windows navegando a:
# http://localhost:8888
Seguridad importante

Si planeas acceder a Jupyter desde otras máquinas en tu red, asegúrate de configurar una contraseña fuerte y considera usar HTTPS. Para uso local, la configuración básica es suficiente.

Usar VS Code con Jupyter en WSL

La mejor experiencia es combinar VS Code con WSL y Jupyter:

  1. Instalar extensión Remote-WSL en VS Code
  2. Conectar VS Code a WSL: Ctrl+Shift+P → "Remote-WSL: New WSL Window"
  3. Instalar extensiones Python y Jupyter en el entorno WSL
  4. Crear notebooks que se ejecutarán en el entorno Linux
Abrir VS Code desde WSL WSL Terminal 
# Desde el directorio de tu proyecto en WSL:
mkdir mi_proyecto_jupyter
cd mi_proyecto_jupyter

# Activar entorno virtual:
source ~/jupyter_env/bin/activate

# Abrir VS Code conectado a WSL:
code .

Primeros pasos con Jupyter Notebooks

Una vez que tengas Jupyter configurado, es importante conocer los conceptos básicos para trabajar eficientemente:

Tipos de celdas

  • Code: Contiene código Python ejecutable
  • Markdown: Texto formateado con sintaxis Markdown
  • Raw: Texto sin procesar

Atajos de teclado importantes

Atajos en modo comando (Esc) Keyboard Shortcuts 
A          - Insertar celda arriba
B          - Insertar celda abajo
DD         - Eliminar celda
M          - Cambiar a Markdown
Y          - Cambiar a Code
C          - Copiar celda
V          - Pegar celda
Shift+↑/↓  - Seleccionar múltiples celdas
Atajos en modo edición (Enter) Keyboard Shortcuts 
Shift+Enter - Ejecutar celda y pasar a la siguiente
Ctrl+Enter  - Ejecutar celda sin cambiar selección
Alt+Enter   - Ejecutar celda e insertar nueva abajo
Tab         - Autocompletar
Shift+Tab   - Mostrar documentación

Ejemplo práctico: Tu primer notebook

Crea un nuevo notebook y prueba estos ejemplos:

Celda 1: Importar librerías Python (Code Cell) 
# Importar librerías básicas
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

print("¡Jupyter está funcionando!")
print(f"NumPy versión: {np.__version__}")
print(f"Pandas versión: {pd.__version__}")
Celda 2: Crear datos y visualizar Python (Code Cell) 
# Crear datos de ejemplo
x = np.linspace(0, 10, 100)
y = np.sin(x)

# Crear gráfico
plt.figure(figsize=(10, 6))
plt.plot(x, y, 'b-', linewidth=2, label='sin(x)')
plt.xlabel('x')
plt.ylabel('sin(x)')
plt.title('Gráfico de la función seno')
plt.grid(True, alpha=0.3)
plt.legend()
plt.show()

Buenas Prácticas para Jupyter Notebooks

Para trabajar eficientemente con Jupyter Notebooks, sigue estas recomendaciones:

Estructura y organización

  • Títulos claros: Usa celdas Markdown para crear secciones bien definidas
  • Explicaciones: Documenta tu código con texto explicativo
  • Celdas pequeñas: Mantén las celdas de código cortas y enfocadas
  • Orden lógico: Estructura el notebook de forma que pueda ejecutarse de arriba a abajo

Herramientas Adicionales

Compartir notebooks online

  • GitHub: Renderiza automáticamente los notebooks
  • NBViewer: Servicio gratuito para visualizar notebooks públicos
  • Google Colab: Ejecutar notebooks en la nube gratuitamente
  • Kaggle Notebooks: Para competencias de ciencia de datos
  • Azure Notebooks: Servicio en la nube de Microsoft
Google Colab como alternativa

Si no puedes instalar Jupyter localmente, Google Colab ofrece un entorno Jupyter gratuito en la nube con GPU y TPU disponibles para machine learning.
