Instalación de Python en Windows

Guía paso a paso para configurar Python en tu sistema Windows

Requisitos del Sistema

Antes de instalar Python, asegúrate de que tu sistema cumple con los siguientes requisitos:

  • Sistema Operativo: Windows 7 SP1 o superior (recomendado Windows 10/11)
  • Arquitectura: 32-bit o 64-bit (recomendado 64-bit)
  • Espacio en disco: Al menos 100 MB de espacio libre
  • RAM: Mínimo 512 MB (recomendado 2 GB o más)
Verificar la arquitectura de tu sistema

Para conocer si tu Windows es de 32 o 64 bits, ve a Configuración > Sistema > Acerca de y busca "Tipo de sistema".

Paso 1: Descargar Python

El primer paso es descargar la versión más reciente de Python desde el sitio web oficial:

  1. Ve al sitio web oficial: https://www.python.org
  2. Haz clic en "Downloads" en el menú principal
  3. La página detectará automáticamente tu sistema operativo y mostrará la versión recomendada
  4. Haz clic en el botón "Download Python 3.x.x" (donde x.x es la versión más reciente)
Versiones de Python

Te recomendamos descargar la versión estable más reciente de Python 3. Evita Python 2.7 ya que su soporte terminó en 2020.

Tipos de instaladores disponibles

  • Windows installer (64-bit): Para sistemas de 64 bits (recomendado)
  • Windows installer (32-bit): Para sistemas de 32 bits
  • Windows embeddable package: Versión portable de Python
  • Windows executable installer: Instalador ejecutable tradicional

Paso 2: Ejecutar el Instalador

Una vez descargado el archivo, sigue estos pasos para instalar Python:

  1. Ejecuta el instalador haciendo doble clic en el archivo descargado
  2. ¡IMPORTANTE! Marca la casilla "Add Python to PATH" en la parte inferior de la ventana
  3. Selecciona "Install Now" para una instalación rápida, o "Customize installation" para opciones avanzadas
¡Muy Importante!

No olvides marcar la opción "Add Python to PATH". Esto te permitirá ejecutar Python desde cualquier ubicación en la línea de comandos.

Instalación Personalizada (Opcional)

Si eliges "Customize installation", podrás configurar:

  • Optional Features:
    • Documentation: Documentación offline de Python
    • pip: Gestor de paquetes de Python (recomendado)
    • tcl/tk and IDLE: Entorno de desarrollo integrado
    • Python test suite: Suite de pruebas de Python
    • py launcher: Lanzador de Python para Windows
  • Advanced Options:
    • Install for all users: Instalar para todos los usuarios del sistema
    • Associate files with Python: Asociar archivos .py con Python
    • Create shortcuts for installed applications: Crear accesos directos
    • Add Python to environment variables: Añadir Python al PATH
    • Precompile standard library: Precompilar la biblioteca estándar
Recomendación para principiantes

Si eres nuevo en Python, te recomendamos usar "Install Now" que incluye todas las opciones necesarias por defecto.

Paso 3: Verificar la Instalación

Después de completar la instalación, es importante verificar que Python se instaló correctamente:

Verificar Python en CMD/PowerShell Command Line 
# Abrir CMD o PowerShell y ejecutar:
python --version

# También puedes usar:
python -V

# Para verificar pip:
pip --version

Si la instalación fue exitosa, deberías ver algo similar a:

Salida esperada Output 
C:\Users\Usuario> python --version
Python 3.11.5

C:\Users\Usuario> pip --version
pip 23.2.1 from C:\Users\Usuario\AppData\Local\Programs\Python\Python311\Lib\site-packages\pip (python 3.11)

Probar el intérprete de Python

También puedes probar el intérprete interactivo de Python:

Ejecutar el intérprete de Python Interactive 
# En CMD o PowerShell, escribe:
python

Solución de Problemas Comunes

Error: "python no se reconoce como comando"

Este error indica que Python no está en el PATH del sistema. Soluciones:

  1. Reinstalar Python marcando la opción "Add Python to PATH"
  2. Añadir manualmente al PATH:
    • Ve a Panel de Control > Sistema > Configuración avanzada del sistema
    • Haz clic en "Variables de entorno"
    • En "Variables del sistema", busca "Path" y haz clic en "Editar"
    • Añade la ruta de instalación de Python (ej: C:\Users\Usuario\AppData\Local\Programs\Python\Python311\)
    • También añade la carpeta Scripts (ej: C:\Users\Usuario\AppData\Local\Programs\Python\Python311\Scripts\)

Múltiples versiones de Python

Si tienes varias versiones de Python instaladas, puedes usar el Python Launcher:

Usar Python Launcher py command 
# Ver versiones disponibles:
py -0

# Ejecutar una versión específica:
py -3.11
py -3.10

# Ejecutar la versión más reciente de Python 3:
py -3
Python Launcher

El Python Launcher (py.exe) es una herramienta que viene con Python para Windows y facilita el manejo de múltiples versiones de Python en el mismo sistema.

Editores de Código Recomendados

Aunque Python viene con IDLE, existen editores más potentes para el desarrollo:

Para Principiantes

  • IDLE: Viene incluido con Python, simple y fácil de usar
  • Thonny: Diseñado específicamente para aprender Python
  • Visual Studio Code: Editor gratuito con extensiones para Python

Para Desarrollo Profesional

  • PyCharm: IDE completo para Python (Community Edition gratuito)
  • Visual Studio Code: Con extensión de Python
  • Sublime Text: Editor ligero y rápido
  • Atom: Editor de código abierto
Recomendación

Para este curso, recomendamos Visual Studio Code con la extensión de Python, ya que es gratuito, potente y ampliamente utilizado en la industria.
